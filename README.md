## Setup Instructions
1. **Prerequisites**  
   - Node.js and npm installed  
   - N8N installed and running  

2. **Clone the Repository**  
   ```bash
   git clone https://github.com/odeliyach/job-search-n8n.git
   ```  

3. **Configuration**  
   - Add required environment variables for N8N to connect to APIs.
   - Include necessary API keys for job platforms.
